# Basic Expo App - Quickstart Template

A clean, production-ready Expo app setup with NativeWind (Tailwind CSS) that you can use to quickly start new projects. This template includes:

- **Expo Router** - File-based routing
- **NativeWind** - Tailwind CSS for React Native
- **TypeScript** - Full type safety
- **Modern React** - Latest React 19 with React Compiler
- **New Architecture** - Expo's new architecture enabled

## Quick Start

1. **Install dependencies**

   ```bash
   npm install
   ```

2. **Start the development server**

   ```bash
   npx expo start
   ```

## Running on iOS Simulator

To run the app on iOS Simulator:

1. **Make sure you have Xcode installed** (macOS only)

2. **Start the Expo dev server**

   ```bash
   npx expo start
   ```

3. **Press `i` in the terminal** to open iOS Simulator, or run:

   ```bash
   npm run ios
   ```

   This will automatically:

   - Launch the iOS Simulator (if not already running)
   - Build and install the app
   - Start the development server

## Other Platforms

- **Android Emulator**: Press `a` in the terminal or run `npm run android`
- **Web**: Press `w` in the terminal or run `npm run web`
- **Expo Go**: Scan the QR code with the Expo Go app on your device

## Development

Start developing by editing files in the **app** directory. This project uses [file-based routing](https://docs.expo.dev/router/introduction).
